# 🛡️ Quarantine Protocol

This protocol defines how unverified or potentially dangerous information is handled.

## 1. Landing Zone
All new PDFs and links must first enter `BODY/SKELETON/BONES/DOCS/ORIGIN/`.

## 2. Extraction Mode
- Researcher subagent reads the file in "Isolated Context."
- No external tool access is allowed during extraction.

## 3. Cleansing
- Remove any PII or unneeded bloat.
- Translate raw data into the Source Card format.

## 4. Promotion
- Data only enters `BODY/EYES/SOURCES` after passing the `source_card.schema.yaml` validation.

---
*Safety is the default state of the Immune system.*
